Testing

  • You may select either of the two canvases

    1. myCanvas

    2. myCanvas2

  • To select the canvas and draw the rectangles, fire up the console and type

    Rectangle.associateWithCanvas("myCanvas");
    or
    Rectangle.associateWithCanvas("myCanvas2");

  • There are 6 rectangles numbered 0(red - 10, 10), 1(blue - 20, 20), 2(green - 5, 30), 3(violet - 30, 10), 4(black - 40, 20), 5(orange - 50, 30).

    • Pick any of the rectangles and move it by using the following syntax

      myRects[rectangle number].moveTo(x,y);

      Where rectangle number can take any value from 0-5

      For example,

      myRects[4].moveTo(90,90);

      will translate rectangle 4(black) to the new position (90,90) from old position(40, 20)

  • Understanding the numbers below, after each move operation:
    • First set, the numbers will be in the range 0-5 which shows all the rectangles affected before the move.
    • Second set, the numbers will be in the range 0-5 which shows all the rectangles affected after the move.
